# Working

居家办公摸鱼神器：防止电脑休眠，保持企业微信、QQ 等 IM 在线，空闲时自动调暗显示器省电。

![image](https://github.com/ssmmtt/Working/assets/16244320/c101bea7-f651-4139-9d3c-9d2c2ad567c9)

## 功能

### 防休眠 / 保持 IM 在线

- 开启后每 50 秒模拟 **F15** 按键，刷新系统空闲计时
- 阻止电脑进入睡眠，降低 IM 显示「离开」的概率
- **手动锁屏后失效**

### 空闲自动调暗亮度

- 空闲时将显示器亮度调至最低，检测到键盘或鼠标活动后**自动恢复原亮度**
- **同时支持外接显示器与笔记本内置屏，多显示器全部调暗**：
  - 外接显示器：通过 **DDC/CI**（需显示器支持，OSD 中开启 DDC/CI）
  - 笔记本内置屏：通过 **WMI**
- 不论台式机多显示器、纯笔记本，还是笔记本叠加任意数量外接屏，都会一并调暗 / 恢复
- 调暗阈值跟随 Windows 电源设置中的「关闭屏幕」时间；未设置或无法读取时，默认 **30 分钟**
- 仅在工作时间段内生效（可配置开始 / 结束时间），离开时间段自动恢复亮度

### 系统托盘

- 点击关闭按钮最小化到托盘，不退出程序
- 托盘图标：**红色** = 已开启，**蓝色** = 已关闭
- 支持启动后自动最小化到托盘（`config.ini` 中 `auto_mini=true`）

## 使用

1. 安装 [.NET 8 运行时](https://dotnet.microsoft.com/download/dotnet/8.0)
2. 双击 `Working.exe`，点击 **Working** 按钮开启
3. 配置保存在程序目录下的 `config.ini`

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `auto_run` | 启动时自动开启 | `true` |
| `auto_mini` | 启动后最小化到托盘 | `true` |
| `start_time` | 工作开始时间 | `0:00:00` |
| `end_time` | 工作结束时间 | `23:59:59` |

## 调试日志

在 `AppLog.cs` 中将 `Enabled` 设为 `true` 可开启控制台日志；VS 调试时在「输出」窗口查看。

## 构建

```powershell
./build.ps1
```

输出 `bin/Working/Working_x86.exe` 与 `Working_x64.exe`，绿色免安装，体积约几百 KB。

## 说明

- 外接显示器需支持 **DDC/CI** 才能自动调暗（可用 [Monitorian](https://github.com/emoacht/Monitorian) 测试）
- 笔记本内置屏通过 **WMI** 调暗，无需 DDC/CI；少数机型驱动不暴露 WMI 亮度接口时可能无效
- 调低亮度可一定程度省电，但不如关闭背光省电明显；本方案优先兼顾 **IM 在线** 与 **显示器寿命**
